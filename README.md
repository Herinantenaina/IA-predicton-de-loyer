# IA De prédiction de loyer
Un projet d’intelligence artificielle conçu pour prédire le prix ou la demande de locations immobilières en fonction de la superficie, les toilletes et docuhes(intérieur ou extérieur), du quartier, de l'accessibilité, de l'état, meublé ou non et enfin le nombre de chambre
